# BANKAY-сайт — как собрать и посмотреть

Astro-блог BANKAY (трек B фабрики контента). 2 статьи уже сгенерированы фабрикой (Тот, голос §8, гуманайзер 0 банов).

## На машине с Node 18+ (твой ноут)
```bash
cd bankay-site
npm install
npm run dev        # локально → http://localhost:4321
# или продакшн-сборка:
npm run build      # → dist/  (проверка, что всё компилится)
npm run preview
```

## Что внутри
- `src/content/blog/niacinamide-chto-eto.md` — статья про ниацинамид (5%)
- `src/content/blog/retinol-doma-bez-razdrazheniya.md` — статья про ретинол
- `src/pages/index.astro` — главная (брендинг BANKAY)
- sitemap + RSS + `public/llms.txt` (для SEO и ИИ-поиска)
